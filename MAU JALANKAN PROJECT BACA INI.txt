
Project ini menggunakan Framework Laravel
jadi untuk menjalankan project ini anda harus mengikuti langkah berikut :

1. Pertama install composer 
   Link : https://getcomposer.org/download/

2. install install githbash 
   Link : https://git-scm.com/downloads

3. install laravel di githbash menggunakan perintah
    - composer global require laravel/installer

4. kemudian buka folder projectnya, buka menggunakan gitbash
   - cd Project-Listbuku

5. jalankan project laravelnya, ketik di terminal githbash nya
   -php artisan serve

5. buka control panel XAMPP dan jalankan Apache dan Mysql

6. buka local database atau localhost/phpmyadmin/

7. buat database baru dengan nama listbuku

8. migrasi data yang ada di laravel ke database, ketik perintah di gitbashnya
   -php artisan migrate:fresh --seed

9. buka link local domain dari project laravelnya.
